# 📚 StudyHub

> **A simple, modern, and responsive Notes Management App built with the MERN Stack.**
>
> ✨ Create • Edit • Delete • Organize your notes effortlessly.

🌐 **Live Demo:** [https://studyhub-w1x6.vercel.app/)](https://studyhub-w1x6.vercel.app/)

---

## 🚀 Overview

StudyHub is a clean and intuitive notes application designed to make note-taking quick and hassle-free. It allows users to create, edit, and delete notes in real time with a responsive and user-friendly interface.

Whether you're managing study notes, daily tasks, or personal reminders, StudyHub keeps everything organized in one place.

---

## ✨ Features

* 📝 Create new notes instantly
* ✏️ Edit existing notes
* 🗑️ Delete notes
* ⚡ Real-time updates
* 📱 Fully responsive design
* 🎨 Clean and minimal UI
* 🔗 REST API integration
* ☁️ Backend connected with MongoDB

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/studyhub.git
```

### Go to the project folder

```bash
cd studyhub
```

### Install dependencies

Frontend

```bash
npm install
```

Backend

```bash
npm install
```

### Start the backend

```bash
npm start
```

### Start the frontend

```bash
npm run dev
```
